* URL encoding/decoding

#+BEGIN_SRC python
  >>> s = u'1234567890-/:;()$&@".,?!\'[]{}#%^*+=_\|~<>\u20ac\xa3\xa5\u2022.,?!\''
  >>> urllib2.quote(s.encode("utf8"))

#+END_SRC
